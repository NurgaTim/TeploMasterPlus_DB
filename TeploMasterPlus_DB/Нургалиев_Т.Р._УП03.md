\*\*Группа:\*\* ИСП(с)3323  

\*\*Предприятие:\*\* ООО «ТепломастерПлюс» (окна и отопление)  



\---



\## Описание проекта



Информационная система предназначена для учёта работы сотрудников компании, занимающейся установкой и ремонтом окон, а также монтажом и обслуживанием систем отопления.



Система позволяет фиксировать отработанные часы каждого сотрудника, указывать дату работы и тип выполняемых работ (окна или отопление). Все данные сохраняются в базе данных SQL Server, что обеспечивает их сохранность после закрытия программы.



\---



\## Функционал системы



| Функция | Описание |

|---------|----------|

| \*\*Добавление записи\*\* | Выбор сотрудника из списка, указание даты и количества отработанных часов. Запись автоматически сохраняется в базу данных. |

| \*\*Удаление записи\*\* | Выбор записи в таблице и её удаление с подтверждением действия. |

| \*\*Просмотр записей\*\* | Все записи отображаются в таблице с автоматической нумерацией (1, 2, 3...). |

| \*\*Очистка таблицы\*\* | Удаление всех записей из базы данных одной кнопкой. |

| \*\*Хранение данных\*\* | Данные хранятся в базе данных SQL Server. После закрытия программы записи не пропадают. |

\## Структура базы данных



База данных `TeploMasterPlus\_DB` содержит 4 таблицы:



| Таблица | Назначение |

|---------|------------|

| \*\*Сотрудники\*\* | Список сотрудников (ФИО, должность, телефон, дата найма, активен) |

| \*\*ЗаписиОРаботе\*\* | Записи о работе (сотрудник, дата, часы, тип работы, объект, подтверждено) |

| \*\*ТипыРабот\*\* | Справочник типов работ (Окна, Отопление, Выезд и т.д.) |

| \*\*Объекты\*\* | Адреса и клиенты (адрес, клиент, телефон, дата создания) |



\*\*Связи между таблицами:\*\*

\- `ЗаписиОРаботе.НомерСотрудника` → `Сотрудники.Номер`

\- `ЗаписиОРаботе.НомерТипаРаботы` → `ТипыРабот.Номер`

\- `ЗаписиОРаботе.НомерОбъекта` → `Объекты.Номер`

\## Пример работы программы



1\. При запуске открывается главное окно с пустой таблицей.

2\. Пользователь выбирает сотрудника из выпадающего списка.

3\. Указывает дату работы и количество часов.

4\. Нажимает кнопку «Добавить» — запись появляется в таблице с новым порядковым номером.

5\. Для удаления записи нужно выделить её в таблице и нажать «Удалить».

6\. Кнопка «Очистить всё» удаляет все записи из базы данных.
